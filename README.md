Repository for the group project on spin dynamics caused by the dipolar coupling between spins. For the QIST4300 Qubit Dynamics and Quantum Control course at TU Delft.

The final presentation can be found [here](https://docs.google.com/presentation/d/100-9EL0gzmZPG6ab2sW0C7LZ5ZcQitrTK_JCwmUltBo/edit?usp=sharing).

We used the following as references for the project:
1. [https://qutech.nl/wp-content/uploads/2017/03/Decoherence-dynamics-of-a-single-spin-versus-spin-ensemble.pdf](https://qutech.nl/wp-content/uploads/2017/03/Decoherence-dynamics-of-a-single-spin-versus-spin-ensemble.pdf)
2. [https://arxiv.org/abs/0904.0263](https://arxiv.org/abs/0904.0263)
